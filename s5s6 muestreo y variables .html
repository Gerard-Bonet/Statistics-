# Para ilustrar dos tcnicas de muestreo aleatorias, sistemtica y simple, vamos a quedarnos con tan slo tres variables, 
# una categrica, y dos numricas: 
# preferred_foot, agility, crossing.


dff= df [["preferred_foot", "agility", "crossing"]]

Nuestro punto de partida ser un muestreo piloto lo suficientemente amplo para calcular la desviacin estandar de cada variable numricas, as como los porcentajes de la categrica. Supondremos que la nica informacin de la que dispondremos ser la poblacin total, N = len (dff), por lo que vamos a ignorar toda la informacin del data set que no obtengamos de las muestras, salvo N

Una vez, estimdas las dos desviaciones, y los porcentajes de la variable categrica, las usaremos como informacin pasada, para estimar el tamao de la muestra, n. Por cada variable habr un nmero de muestra mnimo dentro de un coeficiente de confianza, inventado, y un error relativo mximo. Para ello sacaremos el cclulo del Estimador insesgado de la Media de la muestra, y de la proporcin de clase.

Teoras explicadas y demostradas en los siguientes enlaces. http://matematicas.unex.es/~inmatorres/teaching/muestreo/assets/Cap1.pdf http://matematicas.unex.es/~inmatorres/teaching/muestreo/assets/Cap_3.pdf http://matematicas.unex.es/~inmatorres/teaching/muestreo/assets/cap_5.pdf

Llegando a dos frmulas.

  • \begin{aligned}\cdot \\ n=\dfrac{NZ^{2}\sigma ^{2}}{\left( N-1\right) E^{2}+z^{2}\sigma ^{2}}\end{aligned}
  • \begin{aligned}\cdot \\ n=\dfrac{Z^{2}Npq}{\left( N-1\right) e^{2}+Z^{2}pq}\end{aligned}

def mediana (z):# ha de ser un df de una sola columna 
    z= z.to_numpy()
    y =z.T
    x=np.sort(y)
   
    if len(x )%2 != 0:
        return x[len(x)//2+1]
    else : 
        return (x[len(x)//2]+ x[(len(x)//2)+1])/2
    
def moda (x):# x ha de ser un dataframe de una sola columna
    y1= x.value_counts()

    return y1[y1==y1.max()].index.to_list()

def media (x): # x ha de ser un dataframe de una sola columna
        y= x.to_numpy()
        z= y.T
        w= np.nansum(z)
        return w/len(z)

# definimos n1,n2,n3 como los tamaos de la muestra mnimos, pata un coeficiente de confianza del 70 % con un margen de error 
# relativo del 10 %, tanto si tienen una tendencia normal o no, podemos usar el cuantil de la media Z( va normal o desigualdad de
# Chebyshev si no cumple con una tendencia a la media). Esto nos dice que para un tamao n de una muestra, hay un 70% de que 
# las propiedades de la poblacin total, se encuentren representadas en la muestra 
# Z para una confianza del 70 % es de 0,53, y el error relativo E = 0.1
E= 0.1
Z= 0.53
N = len (dff)
# sumando al resto de variables para calcular el tamao de la muestra , N, D1, D2,p, q
# repasando la frmula de la extraccin del tamao de la muestra .
  • \begin{aligned}\cdot \\ n=\dfrac{NZ^{2}\sigma ^{2}}{\left( N-1\right) E^{2}+z^{2}\sigma ^{2}}\end{aligned}
  • \begin{aligned}\cdot \\ n=\dfrac{Z^{2}Npq}{\left( N-1\right) e^{2}+Z^{2}pq}\end{aligned}
